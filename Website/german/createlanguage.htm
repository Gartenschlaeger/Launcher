Sprache erstellen
Falls Sie ihre Sprache nicht in der Auswahl finden, so können Sie eine eigene erstellen.

Am besten Sie laden Sich dazu das Zusatzprogramm Language Translator runter, und kopieren es in ihrem
Launcher Programmverzeichnis, öffnen Sie dann eine vorhandene Sprachdatei, übersetzen Sie diese,
und speichern Sie die Datei unter einen neuen Name ab.
Alternativ können Sie das ganze auch manuel wie folgt machen.
Gehen Sie dazu im Unterordner Launcher/Language/ und duplizieren Sie eine vorhandene Sprachdatei.
Benennen Sie die neue Datei um in Sprachname.lng.
Sprachname steht hierbei für den Name ihrer Sprache der auch später in der Auswahl erscheinen wird.
Öffnen Sie die Datei mit einen gewöhnlichen Texteditor und übersetzen Sie diese.
Speichern sie ab, und sofort sollte Launcher diese Sprache unter Einstellungen und Oberfläche/Sprache anzeigen.

Bitte senden Sie mir neue Sprachdateien zu, somit können auch andere diese nutzen.
Selbstverständlich dürfen Sie ihr Copyright in die Sprachdatei schreiben.

Senden Sie die Datei einfach an der Supportadresse.
