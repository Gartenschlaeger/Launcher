Verzeichnisse
Hier können Sie die Verzeichnisse und Suchmaschiene fürs Internet festlegen.



Eigene Dateien
Hiermit legen Sie den Path zu Eigene Dateien fest.
Mit Maximiert öffnen können Sie festlegen, das Eigene Dateien immer maximiert geöffnet werden sollen.

Ordnernamen
Hier können Sie die einzelnen Namen der Standard-Ordner unter Eigene Dateien festlegen.
Launcher wird diese Ordner automatisch erstellen sobald Sie die Einstellungen übernehmen, alte Ordner müssen manuell gelöscht werden.

Internet Browser Startseite
Hiermit legen Sie die Startseite bzw Suchmaschiene für die Option Suchen/Im Internet fest.
Diese Seite wird aufgerufen, sobald Sie auf Suchen/Im Internet klicken.
Falls Sie den Internet Browser immer maximiert öffnen möchten, so aktivieren Sie die Option Maximiert öffnen.
