Starting Events
Under Starting Events you can appoint all Settings for the Launcher's Start.



Display at Start
Fade in Launcher's Mainwindow at Start.

Check Version
Launcher check at Start if a new Version is available.

Check OS
Check at start the Operation System Compatibility with Launcher.

Check Programs
At Launcher-Start the Database of Programs will be checked for incorrect inputs

Back up Preferences
With this Option Launcher will create a Backup-Copy of the Preferences (Launcher.cfg).
This Backup-Copy will save in the same Folder as the Prefences.
With further settings you can define when Launcher makes a Bakcup-Copy..

Desktop Wallpaper
Here you can direct Launcher to change the Desktop-Wallpaper at the start.
Please note, that only the Windows-Bitmap-Format will be assisting.
