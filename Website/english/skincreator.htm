SkinCreator


The SkinCreator is a Add-On-Program for Launcher,
with its Help can you convert Skinressourcen into ready skins.
SkinCreator is find under Miscellaneous/SkinCreator in Mainwindow.

To create a Skin, only give up your Ressource Directory as well as a place to save.
After that you can start with Create the Process of conversion.
Errors only can be display by activate Log-Option.
