Create Skin
Owing to the Skinassistance you're able to configure Launcher at your wishes.

So you can create a own Skin:

Best you take a look at the standardskin 'Blue Dream'.
This you'll found in folder 'Launcher/Skin/' or here.
With Photoshop you're able to open the 'BlueDream.psd'-file,
and consider how the Skin is building.
With the using of the Slice-Tools it is very easy to save the simple pieces.

Otherwise you could use alternate the 'template.bmp',
it contains the size of the several elements of the skin.

By using of the addon 'SkinCreator', which you'll find under 'Other/SkinCreator' in Main-Window,
you could convert all the single files to a skinfile.
It have to be in the folder 'Launcher/Skin/' and must have the extension *.skn.

If you will have created a new Skin, so you could send this to the Support-Adresse.
