Programadministration
With Launcher you can manage your programs very comfortably.
Therefor open the preferences and go to the Programadministration.
The Preferences can be opening with the Contextmenu from Systray-Icon
or with the Button Preferences in Main-Window



Add new Program
1. You appoint a new Programentry
2. Prompt the Programpath.
By manual entry you can use variable values.
The program has to be, beside variables like %windir%,
always on a portable drive.
3. Input a Name, which will be displaying in the menu.
4. Input programparameters, if they will be needing
5. Select the working directory
6. Change the symbol at your pleasure.
7. Define start action and autostart options

For chancing preferences of an existing program, click on the entry and change it.
All grograms with the option autostart, will be open automaticly at Launchers start.

The following variables can be use for programs, parameter, symbol and working directory:

Variable Value
%usbdrive% Alphabetic Character of the Portable Drive
%exedir% Launcher's Home Directory
%windir% Windows Directory
%sysdir% Windows System-Directory
%tempdir% Temporary Directory
%homedir% Profile Directory of the actually notified User
%username% Name of the actually notified Computer-User
