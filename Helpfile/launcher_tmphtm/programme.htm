Programmverwaltung
   
In Launcher können alle Programme komfortable verwaltet werden.
Öffnen Sie dazu die Einstellungen und gehen Sie auf Pogrammverwaltung.
Die Einstellungen können über das Kontextmenü im Systray Bereich oder über dem Button Einstellungen im Hauptfenster geöffnet werden.



  
 
Neues Programm erstellen
 
 
1.
Erstellen Sie einen neuen Programmeintrag
2.
Geben Sie den Programmpath an
 
Sie können hier bei manueller Eingabe auch Variable Werte benutzen.
Das Programm sollte sich jedoch ausser bei Variable Werte wie %windir%
immer auf dem Wechseldatenträger befinden.
3.
Geben Sie einen Name ein der im Menü angezeigt werden soll
4.
Geben Sie Programmparameter ein falls diese benötigt werden
5.
Wählen sie das Arbeitsverzeichnis aus
6.
Wechseln sie bei belieben das Symbol
7.
Legen Sie Starttyp und Autostart fest

Um Einstellungen eines vorhandenen Programmes zu ändern,
klicken Sie auf einen eintrag und ändern Sie diese.

Alle Programme mit der Option Automatisch starten, werden beim Start von Launcher mit gestartet.

Die folgenden Variablen können beiProgramm, Parameter, Symbol und Arbeitsverzeichnis benutzt werden.
Variable
Wert
%usbdrive%
Laufwerksbuchstabe des Wechseldatenträgers
%exedir%
Launcher Programmverzeichnis
%windir%
Windows Verzeichnis
%sysdir%
Windows System-Verzeichnis
%tempdir%
Temporäres Verzeichnis
%homedir%
Profile Verzeichnis des aktuell angemeldeten Benutzers
%username%
Name des aktuell angemeldeten Computer Benutzers
