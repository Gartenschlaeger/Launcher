Oberfläche
   
Hier lassen sich Oberflächliche Dinge und das Startverhalten von Launcher einstellen.


Sprache
Hiermit können Sie die Sprache von Launcher festlegen.
Die Sprache hat keinen Einfluss auf den Skin vom Hauptfenster.

Fensterposition
Left fest an welcher Position auf dem Bildschirm Launcher seine Fenster setzen soll.

Bildschirmabstand
Hiermit können sie einen Abstand zwischen Fenster und Bildschirmrand festlegen.
Hat keinen Einfluss bei Position Zentriert.

Transparentz Effekt
Sollten Sie diese Option aktivieren, so wird Launcher das Hauptfenster mit einen Transparents Effekt, Fading, ein und ausblenden.

Bei Start anzeigen
Das Hauptfenster wird beim Start direkt eingeblendet

Version überprüfen
Launcher überprüft beim Start ob Sie die aktuellste Version benutzen.

Betriebssystem überprüfen
Hiermit überprüft Launcher ob Sie ein kompitables Betriebssystem benutzen.

Programme überprüfen
Bei dieser Option wird Launcher beim Start die Programmeinstellungen überprüfen
und eine kurze Fehlermeldung bei Fehlern einblenden.

Speicherplatzanzeige
Hier können Sie festlegen wie die Speicherplatzanzeige im dargestellt werden soll.
