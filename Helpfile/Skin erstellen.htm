Dank der Skinunterstützung können Sie Launcher komplett an ihren Wünschen anpassen.

So können Sie einen eigenen Skin erstellen:

Am besten Sie schauen Sich den Standardskin Blue Dream an.
Diesen finden Sie im Ordner Launcher/Skin/ oder auf der Internetadresse von Launcher.
Mit Photoshop können Sie die Blue Dream.psd-Datei öffnen,
und sich so ansehen wie der Skin aufgebaut ist.
Mit dem Slice-Werkzeug lassen sich die einzelnen Teile kinderleicht abspeichern.

Ansonsten können Sie auch alternativ die template.bmp benutzen,
sie enthält die größen der einzelnen Elemente vom Skin.

Mit Hilfe des Programmes SkinCreator das Sie unter Weiteres/SkinCreator im Hauptfenster finden,
können Sie dann alle einzelnen Dateien zu einer Skindatei konvertieren.
Diese muss sich im Ordner Launcher/Skin/ befinden und die Erweiterung *.skn haben.

Sollten Sie einen neuen Skin erstellen, so können Sie diesen an der Support-Adresse senden.
Ich werde diesen dann hier mit aufnahmen.
