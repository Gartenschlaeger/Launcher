Hier lassen sich Oberflächliche Dinge und das Start verhalten von Launcher einstellen.



Sprache
Hiermit können Sie die Sprache von Launcher festlegen.
Die Sprache hat keinen Einfluss auf den Skin vom Hauptfenster.

Fensterposition
Legt fest an welcher Position auf dem Bildschirm Launcher seine Fenster setzen soll.

Bildschirmabstand
Hiermit können sie einen Abstand zwischen Fenster und Bildschirmrand festlegen.
Hat keinen Einfluss bei Position Zentriert.

Transparents Effekt
Sollten Sie diese Option aktivieren, so wird Launcher das Hauptfenster mit einen Transparents Effekt, Fading, ein und ausblenden.

Immer im Vordergrund Damit befindet sich das Hauptfenster immer im Vordergrund.
Sie sollten diese Einstellung generell aktivieren.

Systray Symbol
Hier können Sie festlegen welches Symbol Launcher im Systray Bereich benutzen soll.
Sie können ein eigenes vorgeben oder Launcher immer das aktuelle der Skindatei benutzen lassen.

Speicherplatz anzeige
Die Speicherplatz anzeige befindet sich auf dem Hauptfenster uns zeigt Speicherplatz des Wechseldatenträger an.
Hiermit können Sie festlegen ob der freie oder benutze Speicherplatz angezeigt werden soll,
sowie den Darstellungs typ.
