Unter Startereignisse können alle Einstellungen zum Start von Launcher vorgenommen werden.



Bei Start anzeigen
Blendet das Hauptfenster von Launcher beim Start ein.

Version überprüfen
Launcher überprüft beim Start ob eine neue Version verfügbar ist.

Betriebssystem überprüfen
Hiermit wird beim Start das Betriebssystem auf Kompatibilität mit Launcher überprüft.

Programme überprüfen
Die Programmdaten werden auf korektheit überprüft.

Einstellungen sichern
Mit dieser Einstellung erstellt Launcher eine Sicherheitskopie der Einstellungen(Launcher.cfg).
Die Sicherheitskopie wird im selben Ordner wie die Einstellungen gespeichert.
Sie können mit den weiteren Einstellungen festlegen wann Launcher eine Sicherheitskopie erstellen soll.

Desktop Wallpaper
Hier können Sie Launcher anweisen das Hintergrundbild des Desktop beim Start zu wechseln.
Bitte beachten Sie das hier lediglich das Windows Bitmapformat unterstützt wird.
