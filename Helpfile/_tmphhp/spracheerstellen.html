Sprache erstellen
   
Falls Sie ihre Sprache nicht in der Auswahl finden,
so können Sie eine eigene erstellen.

Gehen Sie dazu im Unterordner Launcher/Language/ und kopieren Sie am besten eine vorhandene Sprache.
Benennen Sie die Datei um in Sprachname.lng.
Sprachname ist hierbei der Name ihrer Sprache der auch später in der Auswahl angezeigt wird.
Öffnen Sie die Datei mit einen gewöhnlichen Texteditor und übersetzen Sie diese.
Speichern sie, und sofort sollte Launcher diese Sprache unter Oberfläche/Sprache anzeigen.

Bitte senden Sie mir die Sprachdatei, falls Sie eine eigene entwerfen.
Somit können auch andere diese nutzen können.
Selbsterständlich dürfen Sie ihr Copyright in der Sprachdatei schreiben.

Senden an mail@kaispage.de oder auf http://www.kaispage.de die neuste Adresse besorgen.
