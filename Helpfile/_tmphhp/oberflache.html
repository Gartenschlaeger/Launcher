In Launcher können alle Programme komfortable verwaltet werden.
Öffnen Sie dazu die Einstellungen und gehen Sie auf Programmverwaltung.
Die Einstellungen können über das Kontextmenü im Systray Bereich
oder über dem Button Einstellungen im Hauptfenster geöffnet werden.



Neues Programm eintragen
1. Erstellen Sie einen neuen Programmeintrag
2. Geben Sie den Programmpfad an.
Sie können hier bei manueller Eingabe auch Variable Werte benutzen.
Das Programm sollte sich jedoch außer bei Variable Werte wie %windir%
immer auf dem Wechseldatenträger befinden.
3. Geben Sie einen Name ein der im Menü erscheinen soll
4. Geben Sie Programmparameter ein falls diese benötigt werden
5. Wählen sie das Arbeitsverzeichnis aus
6. Wechseln sie bei belieben das Symbol
7. Legen Sie Starttyp und Autostart fest

Um Einstellungen eines vorhandenen Programmes zu ändern, klicken Sie auf einen Eintrag und ändern Sie diese.
Alle Programme mit der Option Automatisch starten, werden beim Start von Launcher automatisch mit gestartet.

Die folgenden Variablen können bei Programm, Parameter, Symbol und Arbeitsverzeichnis benutzt werden.
Variable Wert
%usbdrive% Laufwerksbuchstabe des Wechseldatenträger
%exedir% Launcher Programmverzeichnis
%windir% Windows Verzeichnis
%sysdir% Windows System-Verzeichnis
%tempdir% Temporäres Verzeichnis
%homedir% Profile Verzeichnis des aktuell angemeldeten Benutzers
%username% Name des aktuell angemeldeten Computer Benutzers
